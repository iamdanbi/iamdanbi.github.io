Mini Pygame Project — KSOP CS Curriculum

Introduction

This project was developed as educational content for KSOP (KAIST Science Outreach Program) students — middle and high schoolers learning programming for the first time. The game was designed to teach core game loop logic in a step-by-step, approachable way.

Features

  • — Arrow-key character movement (left/right)
  • — Dodge falling stones, collect jewels
  • — Score tracking with clear UI
  • — Start, How-to-Play, and Game Over screens
