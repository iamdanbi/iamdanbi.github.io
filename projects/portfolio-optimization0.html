Portfolio Optimization via Convex Programming

Overview

This project applies convex optimization techniques — studied in KAIST's Convex Optimization Techniques course (2025, A0) — to the classical problem of portfolio construction. The goal is to find weight allocations across assets that optimize a risk-adjusted objective under realistic constraints.

Two formulations are implemented: mean-variance optimization (Markowitz) and CVaR (Conditional Value-at-Risk) minimization, which is more robust to heavy-tailed return distributions.

Problem Formulation

The mean-variance problem:

minimize    w' Σ w
subject to  μ' w ≥ r_target
            1' w = 1
            w ≥ 0               (long-only)
            ‖w‖₁ ≤ L            (leverage constraint)

where w ∈ ℝⁿ are portfolio weights, Σ is the covariance matrix, and μ is the expected return vector. The efficient frontier is traced by varying the target return.

What's Implemented

  • — Efficient frontier construction across a range of target returns
  • — CVaR (expected shortfall) minimization as an alternative risk measure
  • — Factor exposure constraints (sector neutrality, beta bounds)
  • — Transaction cost penalty via L1 turnover regularization
  • — Covariance shrinkage using the Ledoit-Wolf estimator
  • — Backtesting on historical S&P 500 constituent data
